# MISSION.md (Example)

**Mission Statement**  
Help our NIAID-funded repository implement the Blueprint so that all datasets are properly described with minimal metadata, assigned appropriate PIDs, exposed via machine-readable APIs, correctly cited, and supported by clear outreach resources.

**Success Criteria**
- All new datasets include the minimal schema.org elements from Table 1.
- Persistent identifiers are assigned to datasets, creators, and hosting organizations.
- Metadata is exposed through a standards-compliant API.
- Citation guidance is publicly posted.
- A designated point of contact for training is identified.

**Constraints**
- Repository uses a custom metadata system (not DataCite or Dataverse).
- Small team (3 people) with limited developer time.
- Must maintain compatibility with existing workflows.

**Out of Scope**
- Full migration to a new repository platform.
- Implementation of complex access controls or authentication.
