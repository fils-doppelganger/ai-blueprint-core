# Reference documents

Place **durable HTML cheat sheets** here (`0001-five-areas.html`, `0002-table-1-elements.html`, etc.).

- **Lessons** (`../lessons/*.html`) teach once; learners may not reopen them.
- **Reference** files compress what learners need on repeat visits—Table 1, PID matrix, JSON-LD minimal shape, citation checklist.

Rules:

- Link `../assets/shared.css` for consistent styling.
- Use terminology from `../GLOSSARY.md` once it exists.
- Lessons should anchor-link into these files as they are created.

Create reference pages when a curriculum topic is first mastered, not before the learner has context (see `../references/curriculum-map.md`).
